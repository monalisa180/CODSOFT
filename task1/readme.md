# 🤖 Rule-Based Chatbot

A simple **rule-based chatbot** built using **HTML, CSS, and JavaScript**.
This chatbot responds to user inputs based on predefined conditions using **if-else logic**.

---

## 📌 Project Objective

The goal of this project is to understand the basics of:

* Natural Language Processing (NLP)
* User interaction handling
* Rule-based response systems

---

## 🚀 Features

* 💬 Interactive chat interface
* 🎨 Clean and professional UI design
* 🧠 Rule-based response system
* ⏰ Shows current time and date
* 😂 Fun responses (jokes, greetings, etc.)
* ⚡ Fast and lightweight (no backend required)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 💡 How It Works

* The chatbot takes user input
* Converts it into lowercase
* Matches keywords using `if-else` conditions
* Returns a predefined response

Example:

```
if (input.includes("hello")) {
    return "Hello! How can I help you?";
}
```
