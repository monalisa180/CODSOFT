# Rule-Based Chatbot

## 📌 Project Description
This is a simple rule-based chatbot built using HTML, CSS, and JavaScript.  
It responds to user inputs using predefined rules (if-else conditions).

## 🚀 Features
- User-friendly interface
- Rule-based responses
- Instant reply system
- Enter key support

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open `index.html` in browser
