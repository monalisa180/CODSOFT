# 🎯 Recommendation System (Task 4)

A simple and interactive **Recommendation System** built using **HTML, CSS, and JavaScript**.
This project suggests **Movies, Books, and Products** based on user preferences using a **Content-Based Filtering approach**.

---

## 📌 Features

* 🎬 Movie Recommendations
* 📚 Book Recommendations
* 🛍️ Product Recommendations
* 🔄 Dynamic dropdown (changes based on category)
* ⚡ Instant results
* 🎨 Clean and simple UI

---

## 🧠 Concept Used

This project uses:

### ✅ Content-Based Filtering

Recommendations are made based on the **user’s selected preference (genre/type)**.

Example:

* If user selects **Action Movies** → shows action movies
* If user selects **Self-Help Books** → shows self-help books
