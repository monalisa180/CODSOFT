---

## 🧠 How the AI Works

The AI uses the **Minimax Algorithm**, a recursive decision-making algorithm used in game theory.

### Key Idea:
- AI tries to **maximize its score**
- Player tries to **minimize AI score**
- The algorithm evaluates **all possible moves**

### Scoring:
- AI win → +10  
- Human win → -10  
- Draw → 0  

This ensures the AI always plays the **best possible move**.
